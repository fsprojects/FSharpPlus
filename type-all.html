FSharpPlus

All

This is a wrapper type for booleans, with a specific set of monoidal operations. The contained bools would become true only if both (all) operands are true.

Related Types

  • Any: Similar wrapper, but using the 'any' criteria.

Abstractions

  • Semigroup
  • Monoid

Examples

#r @"nuget: FSharpPlus"

open FSharpPlus
open FSharpPlus.Data

let res1 = All true ++ zero ++ All false
// val res1 : All = All false

let even x = x % 2 = 0

let res2 = [2;4;6;7;8] |> map (even >> All) |> sum
// val res2 : All = All false
val res1: obj
val even: x: int -> bool
val x: int
val res2: obj
