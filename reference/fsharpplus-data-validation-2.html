Validation<'error, 't>

Namespace: FSharpPlus.Data

A 'Validation' is either a value of the type 'error or 't, similar to 'Result'. However, the 'Applicative' instance for 'Validation' accumulates errors using a 'Semigroup' on 'error. In contrast, the Applicative for 'Result' returns only the first error.

A consequence of this is that 'Validation' is not a monad. There is no F#+ 'Bind' method since that would violate monad rules.
