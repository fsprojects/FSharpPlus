State<'s, 't>

Namespace: FSharpPlus.Data

Computation type: Computations which maintain state. Binding strategy: Threads a state parameter through the sequence of bound functions so that the same state value is never used twice, giving the illusion of in-place update. Useful for: Building computations from sequences of operations that require a shared state.
